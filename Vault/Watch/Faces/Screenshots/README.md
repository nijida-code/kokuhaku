# Watch Face Screenshots

This directory contains screenshots corresponding to `.watchface` presets.

## Purpose
- documentation (readme, website, repo pages)
- future in-app galleries (iOS)
- design reviews

## Naming
Use the same base name as the preset, but with an image extension:

- `Kokuhaku_<Name>.png`

Examples:
- `Kokuhaku_Focus.png`
- `Kokuhaku_Minimal.png`

## Notes
- Prefer screenshots that clearly show complication content.
- Avoid personal data in screenshots.
- Keep images reasonably small (compress if needed).
